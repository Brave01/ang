二 Go语言基础
     1 Go语言编译器以及开发工具的安装。
        现在Go语言编译器的最新版本是1.11版本。开发工具可以使用liteIDE、vscode、        sublime。
        windows平台，对于Go语言编译器直接默认安装。安装完成后，在控制台窗口中，
        go version-查看Go编译器版本
        go env-可以查看Go的各种设置，包括各种路径的设置。
        唯一需要我们设置的环境变量是GOPATH。是Go语言项目的源代码路径。例如：
        GOPATH = e:\godev
        然后，在GOPATH目录下，添加一个src子目录，所以的多个项目源码都可以放到src
        目录下。
        src-用户手动添加的目录(存放的Go源码)
        bin-使用go install命令，可以自动生成，存放可执行程序
        pkg-使用go install命令，可以自动生成，存放包的二进制文件，(*.a 库文件) 

        liteIDE的免安装的，解压缩，找到exe程序即可。(与操作系统的机器字长无关)

     2 如果安装好的同学，练习几个命令
        go build -编译程序，在当前目录下生成exe程序，只能在当前目录下执行。
        go run  hello.go-直接执行程序
        go install-安装，把exe程序生成到GOPATH的bin子目录下。这个exe可以在任意的目录
                       下执行
        go get url路径-安装远程包
     3 Go语言程序的组成部分
        3.1 包的声明语句
              package main
              main包包含了main函数，main的包名称可以和所在的目录名称不相同
        3.2 导入要使用的包
              import  "fmt"
              如果是多个包，放到一起导入
              import(
                    "fmt"
                    "time"
                    "os"
              )
             只能导入在代码中使用的包，如果包导入了，但是没有使用，导致语法错误
        3.3 函数
             func main(){
               fmt.Println("Hello 世界")
             }    
             Go语言原生态支持utf-8编码。Go的源文件默认就是采用utf-8编码。
            另外，Go语言的注释与c/c++一样。// 单行注释  /* */多行注释或块注释
        
      练习：自己写一个基本的go文件，使用命令编译、执行程序。
     4 各种不同类型的变量与零值得比较<高质量的c++>变成 林锐 ，附录B。
     5 完成一个计算器，可以+ - * / %平方、开方。
